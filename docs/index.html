Features

  • compare 2 json. also supports json5, json with comments, or plain text
  • json diff as visual diff or json (see delta format)
  • compare json from a gist (example)
    1. create a gist
    2. add 2 json files on the gist
    3. paste the gist url here:
  • smart diff arrays/lists. if items are objects a "key", "id", or "name" is automatically used to match
  • text diff. for long string or text, a text diff is done at characther level (using google/diff-match-patch)

powered by the jsondiffpatch library
